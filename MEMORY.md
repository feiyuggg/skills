### 📊 五维价值投资分析框架

#### 维度1: 投资点位与期权策略
- **52周位置**: 判断当前估值区间 (<30%深度价值区)
- **左侧建仓**: 金字塔法 (20%-30%-30%-20%)
- **Sell Put策略**: IV>历史均值20% + 股息率>2%

#### 维度2: 基本面与护城河 (段永平五维)
1. 网络效应 (用户规模/平台双边市场)
2. 品牌护城河 (定价权/用户忠诚度)
3. 技术护城河 (专利壁垒/R&D)
4. 成本优势 (规模经济/运营效率)
5. 切换成本 (数据沉淀/迁移难度)

#### 维度3: 每股现金流 (FCF/OCF)
- **特殊处理**: Capex激增期用OCF×70%估算
- **P/OCF**: 对比历史均值和同业
- **质量指标**: FCF/净利润 >70%

#### 维度4: Forward P/E与PEG
- **多时间维度**: 1年PEG(短期) + 2年PEG(中期) + 3年CAGR(长期)
- **PEG标准**: <0.8严重低估, 1.0-1.2合理, >1.5高估

#### 维度5: DCF三阶段模型
- **阶段1**: 高成长3-5年 (g1)
- **阶段2**: 中成长3-5年 (g2 = g1×0.6)
- **阶段3**: 永续增长 (g3 = 2-3%)
- **安全边际**: >30%强烈买入, 20-30%建议买入

### 💡 核心投资原则

1. **买公司不是买股票** - 关注企业本质
2. **和时间做朋友** - 长期持有优质企业
3. **左侧交易** - 在别人恐惧时贪婪
4. **安全边际** - 30%以上折扣才重仓
5. **护城河** - 极宽/宽阔护城河是前提
