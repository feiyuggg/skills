# 小天才股票投资分析框架 v1.0

**制定者**: Gaven  
**制定时间**: 2026-02-11  
**投资哲学**: 左侧交易 + 价值投资（巴菲特/段永平）+ 和时间做朋友

---

## 核心原则

1. **买公司不是买股票** - 看企业本质，不看股价波动
2. **别人恐惧时贪婪** - 左侧交易，逆向布局
3. **长期持有** - 和时间做朋友，不看短期涨跌
4. **安全边际** - 只在合理或低估价位买入

---

## 五维分析框架（必须严格执行）

### 维度1: 投资点位和期权策略
- 当前价位评估（52周区间位置）
- 左侧交易入场策略（分4档：第一批、第二批、重仓、极限）
- 每档的仓位建议和安全边际说明
- 一年期期权策略（Sell Put、垂直价差、备兑看涨）
- 每项策略的风险提示和投资建议评级

### 维度2: 前瞻PE + 现金流 + 基本面 + 护城河
- **前瞻PE**: 数值、行业对比、历史均值、投资建议
- **现金流分析**: 净利润率、经营现金流、自由现金流、ROIC、P/FCF
- **基本面五维**: 营收增长、盈利能力、资产负债表、管理层、股东回报
- **护城河分析**（段永平思路）:
  - 网络效应护城河
  - 品牌护城河
  - 技术护城河
  - 成本优势护城河
  - 转换成本护城河

### 维度3: 每股现金流 (FCF per share)
- FCF详细计算过程
- P/FCF估值
- FCF Yield
- 与同行对比
- FCF增长趋势

### 维度4: PEG估值
- PEG计算公式和结果
- PEG评估标准表（<0.8到>2.0的分级）
- 与主要同行对比（MSFT、META、AMZN等）
- 左侧交易理想的PEG买入区间

### 维度5: 未来现金流折现估值 (DCF)
- 三阶段假设（高增长期、中速增长期、永续增长期）
- WACC和增长率假设说明
- 详细DCF计算过程
- 敏感性分析表（不同WACC和增长率组合）
- 基础/乐观/保守三种情景的内在价值
- 左侧交易的理想买入价位（对应安全边际）

---

## 输出格式要求

### 投资评级
- 必须给出明确的评级（BUY/HOLD/SELL）和星级（⭐⭐⭐⭐⭐）

### 左侧交易执行计划
- 总资金分配说明
- 分价位建仓计划（具体价格和仓位百分比）

### 期权策略建议
- 首选策略（详细参数）
- 备选策略
- 每种策略的逻辑、预期收益、风险提示

### 与时间做朋友的持有逻辑
- 用5句话总结长期持有的核心理由
- 3年目标价预测
- 巴菲特式结论
